FRAME Visualise every shot

A storyboard communicates the film before a frame is shot. OPENFRAME gives you a fast, flexible canvas — sketch directly, pick from compositional templates, or describe a shot in words and let AI generate the reference image.

Panel Templates
SVG-based shot compositions — close-up, wide, two-shot, POV, insert and more
Freehand Draw
Draw directly onto any panel. No drawing tablet required — mouse or touchscreen
AI Generation
Type a shot description → AI generates a reference image. Powered by Pollinations.ai — free, no API key
PDF Export
Export the full board as a single PDF — formatted for printing and sharing with crew
